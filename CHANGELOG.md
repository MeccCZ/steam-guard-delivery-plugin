# Changelog

Все значимые изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-04-27

### Добавлено
- ✨ Автоматическая выдача Steam Guard кода при покупке
- ✨ Команда `!code` для запроса кода
- ✨ Ограничение доступа на 30 дней (настраивается)
- ✨ База данных покупателей с отслеживанием срока
- ✨ Управление через Telegram бота
- ✨ Поддержка множества Steam аккаунтов
- ✨ Генерация кодов из shared_secret (локально)
- ✨ Логирование всех операций

### Исправлено
- 🐛 Удаление невидимых Unicode символов из команд FunPay
- 🐛 Корректная обработка сообщений без водяных знаков
- 🐛 Правильная регистрация обработчиков Telegram

### Безопасность
- 🔐 Локальная генерация кодов (shared_secret не передается)
- 🔐 Проверка срока доступа перед выдачей кода
- 🔐 Логирование всех запросов кодов

## [Unreleased]
